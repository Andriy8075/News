# NewsHub

#### A website for creating and reading news. Made by students of group IM-34 Ashomka Andrii and Kurylenko Arthem

## How to launch this website:

1. Open Docker
2. Copy these command and paste them in your local terminal:
```bash
git clone https://github.com/Andriy8075/News.git
cd News/docker/prod
docker-compose up
```
These commands will clone git repo, cd to folder with docker production build and launch docker-compose build.
